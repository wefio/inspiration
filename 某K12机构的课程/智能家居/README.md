# 智能家居
主控板是Arduino UNO，拓展板是Arduino ph2.0拓展板，全套可以看看淘宝的 易创空间。<br>
前5节是原有教程，原教程应该有16节，但是受限于个人能力、主控板性能、电力，只能同时用部分传感器。综合运用是结合前面所有内容的整合实现。<br>
原教程直接使用delay()进行等待，但多传感器不能这么做，这样会阻塞其他传感器，所以必须用非阻塞式设计的计时器millis()方案。<br>
简单来说，非阻塞式设计是：不通过死循环或空转等待来延迟执行，利用定时器/计数器配合状态机实现延时，使得CPU或逻辑电路在等待期间可以处理其他任务。也就是在触发时计时然后用简单的状态机切换到另一状态，计时结束再切换回原状态。所以程序会比前面几节看起来复杂很多。
